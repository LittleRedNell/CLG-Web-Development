What is HTML?

HTML stands for hyper text markup language. HTML is the language used to write webpages. HTML can be described as the structure, content or words on a webpage. All HTML documents must contain a doctype code and are composed of elements (eg head/body elements).

What is CSS?

CSS stands for Cascading Style Sheets. CSS control the presentation of a webpage such as the colours, text size, text style. CSS controls the presentation of specific parts of a webpage by using rules that have a selector, properties and values. CSS is added to a webpage via one of three methods;

  • inline styles
  • internal CSS, or
  • external styleheets.
Inline styles add style attributes to HTML elements. Internal CSS places rules inside a 'style' element in the head section of a webpage. External stylesheets use a link element in the head section that links to an external .css file containing CSS rules. This webpage was styled using an external stylesheet.
